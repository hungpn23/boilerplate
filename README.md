# Nestjs boilerplate

A nestjs boilerplate project

## Tech stack

Nestjs, MySQL

## Installation

Install with pnpm

```bash
  git clone https://github.com/hungpn23/boilerplate.git
  cd boilerplate
  pnpm install
```

Create and configure `.env` file from `example.env`

```bash
  pnpm start:dev
```
